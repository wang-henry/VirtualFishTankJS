t1.renderTank()
t1.addStationaryObj('assets/rock.svg', 0, 0, '70px', '70px')
t1.addStationaryObj('assets/rock.svg', 900, 0, '100px', '100px')
t1.addStationaryObj('assets/rock.svg', 500, 0, '50px', '100px')
t1.addStationaryObj('assets/seaweed.svg', 500, 0, '100px', '100px')

t1.addFish('assets/fish.svg', 0, 0, 20, 0, 50, false)

t1.updateAllHOrientation()

Multiple fish with varying speeds and movement delays

Users can define set horizontal points for the fish to travel between, they can configure the fish to "bounce" between the points or simply loop back. The first at the top is bouncing between the set points, where the fish at the bottom is looping.

Custom fish movement path with custom trails can be used to create animations for things such as loading screens
