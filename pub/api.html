Tank Configuration

Property Default Type Description
width 100 int width of the tank DOM element in pixels
height 100 int height of the tank DOM element in pixels
div N/A DOM div DOM element that the fish tank is a child of (if the developer choses to specify a DOM element that the tank should go under)

Creating a new fish tank example


        // Create a new tank with width, height and using the div named 'tank'
        const fishTank = new Tank(width, height, document.getElementById('tank'))

Tank Functions

Function Arguments Type Description
renderTank() None N/A Makes the tank visible on the page.
changeWater() color str Change the water color of the tank with a given color.
changeBorder color str Change the border color of the tank with a given color.
changeStyle style str Change the border style of the fish tank with a given style.
addFish() fishPath, startX, startY, xSpeed, ySpeed, moveDelay, controlled str, int, int, int, int, int, bool Add a fish with the image located at fishPath to the fish tank. The fish will start at (startX, startY) and move with speeds of xSpeed in the x axis and ySpeed in the y axis when movement is enabled. The movement delay can be specified by moveDelay. If controlled is true then the fish will be user controlled, otherwise it will move around the tank on its own. By default controlled is false.
addStationaryObj() imgPath, x, y, objW, objH str, int, int, int, int Add a stationary object located at the image path imgPath. The object will be located at (x,y) in the tank with height and widths objW and objH.
addText() text, x, y str, int, int Add text to the tank at position(x,y).
enableAllMove() None N/A Enables movement for all fish in the tank.
updateAllFish() xSpeed, ySpeed, moveDelay int, int, int Changes the speeds and movement delay of all fish in the current tank to xSpeed, ySpeed, moveDelay
updateAllStartPos() startX, startY, refresh int, int, bool Change the starting positions of all fish in the tank to (startX, startY). If Refresh is true then all the fish will be set back to their starting positions, by default this is false.
updateAllHOrientation() None N/A Flips the horizontal orientation of all fish in the tank.
updateAllVOrientation() None N/A Flips the vertical orientation of all fish in the tank.
updateHOrientation() index int Flips the horizontal orientation of a particular fish at index. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0.
updateVOrientation() index int Flips the vertical orientation of a particular fish at index. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0.
updateWidth() index, width int, int Update the width in pixels of a particular fish at index. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0.
updateHeight() index, width int, int Update the height in pixels of a particular fish at index. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0.
setTravelPoints() index, x1, x2, yLevel, loop, loopType int, int, int, int, bool, str Set two x travel points x1, x2 and a yLevelfor a particular fish at index to swim between. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0. loop will determine whether this animation should be repeated, by default it is false. loopType can be either 'normal' or 'bounce'. 'normal' means that the fish will teleport back to x1 after reaching x2. 'bounce' means that the fish will swim back and forth between the two points x1, x2.
setTrail() index, color, height int, str, int Set a rectangular trail on the left of the fish at index. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0. This should only be used in conjunction with setTravelPoints, otherwise the trail may not appear. If used with 'bounce' loop type, the fish will push the trail back inward when it turns around.
removeTrail() index int Remove the trail of the fish at index. Index order is determined by the order the fish are initialized, for example the first fish will be counted as index 0.
